# 레일 이동하중 구조검토

DXF 3D 경로를 따라 이동하는 기기(1점 집중하중)가 레일에 가하는 **동적 하중**을
위치별로 산정하고, 3D 프레임 해석으로 **각 지지점 반력**과 **부재 단면력**의
**최대값(포락선)** 과 발생 위치를 찾는 프로그램. 구조설계 안전성 검토(KDS 등)의
입력 하중을 산정하는 것이 목적이다.

## 동작 원리

1. **DXF 파싱** — 3D 폴리라인(레일 경로)과 지정 레이어의 `POINT`(지지점) 추출
2. **경로 기하** — 호장 매개변수화, 노드별 접선·주법선·곡률반경 R 계산
3. **모델 생성** — 경로를 빔요소로 분할, 지지점 좌표에 가장 가까운 노드를 지지
4. **이동하중** — 각 위치에서 동적 하중 산정
   - 연직 자중: `W = m·g`, 충격계수 `DAF = 1 + φ`
   - 접선 관성(가속/제동): `Fₜ = m·a`
   - 곡선 원심력: `F_c = m·v² / R` (주법선 방향)
5. **반복 해석** — 기기를 경로 노드마다 이동시키며 3D 직접강성법으로 해석
   (강성행렬은 한 번만 분해해 위치별 하중에 재사용)
6. **포락선 집계** — 지지점별 최대 반력, 부재별 최대 휨/전단/축력/비틀림과
   발생 위치를 산출

## 설치

```bash
pip install -r requirements.txt
```

(numpy, ezdxf, matplotlib / tkinter는 파이썬 표준)

## 사용

### GUI

```bash
python app.py
```

DXF 선택 → 레이어 지정 → 단면/기기/옵션 입력 → "해석 실행".
3D 경로에 지지점 반력이 색으로 표시되고, 지배 지지점의 반력 영향선이 그려진다.

### 커맨드라인

```bash
python make_sample_dxf.py          # 샘플 DXF 생성
python run_cli.py sample_rail.dxf --mass 2000 --speed 1.5 --accel 0.5 --impact 0.3
```

## 입력 규약

- **경로**: 3D `POLYLINE` / `LWPOLYLINE` / `SPLINE` / 연결된 `LINE` (가장 긴 것 자동 채택, 레이어 지정 가능)
- **지지점**: 지정 레이어(기본 `SUPPORT`)의 `POINT` 객체 — 가장 가까운 해석 노드가 지지점이 됨
- **단위**: 입력은 E[GPa], A[cm²], I/J[cm⁴], 질량[kg], 속도[m/s], 가속[m/s²]. 내부는 SI, 결과는 kN·kN·m

## 가정 / 한계 (v1)

- 기기는 **1점 집중하중**으로 단순화 (다점 대차는 추후)
- 속도·가속은 경로 전 구간 일정으로 보고 곡률에 따른 원심력을 위치별 계산 → **최악 포락선**
- 지지조건 프리셋: `pinned`(병진 구속) / `vertical`(연직만) / `fixed`(완전고정)
- **KDS 강도 검토(허용응력 비교)는 후속 단계** — 현재는 하중·단면력 산정까지
